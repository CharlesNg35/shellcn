# Phase 2c · Step 5 — Desktop/graphical recording framework

**Milestone:** M1.6 · **Status:** ☐ Not started
**Depends on:** 2c.1, 2c.2, 2c.3 · **Spec refs:** [v2 §9.5](../../v2.md), [v2 §11](../../v2.md), [v2 §12](../../v2.md)

## Goal
Define a protocol-neutral desktop recording path for VNC now and RDP/Guacamole
later, while being explicit about trust boundaries.

## Checklist
- [ ] Define desktop recording formats: `webm_canvas` (browser capture), `guacamole_native` (future sidecar), and future `rfb_native`
- [ ] Browser path: remote desktop panel can capture its canvas via `HTMLCanvasElement.captureStream()` and encode chunks via `MediaRecorder`
- [ ] Chunk APIs: start, upload chunk, finalize, abort; all tied to a server-created recording id and CSRF/session auth
- [ ] Plugin-native path: optional plugin `RecordingProvider` can write authoritative desktop recordings through the core sink
- [ ] Mark browser-captured desktop recordings as non-authoritative in metadata; do not use them for strict compliance claims
- [ ] Playback: WebM via native `<video>` first; plugin-native formats can expose a converter/player later
- [ ] Handle browser support/mime fallback; if unsupported and recording is required, deny stream before connection

## Definition of done
The platform has a generic desktop recording contract. A noVNC canvas can be recorded to WebM when enabled, and a plugin can later provide a native recorder without changing the recording list/playback UI.

## Tests (gate)
- [ ] Frontend unit test with mocked `captureStream`/`MediaRecorder` uploads chunks and finalizes
- [ ] Server tests validate chunk ordering, max chunk size, abort/finalize transitions, and authz
- [ ] Required desktop recording with unsupported browser produces a clear pre-stream error
- [ ] Metadata distinguishes `webm_canvas` from plugin-native/authoritative formats

## Notes
Desktop protocols vary heavily. The core owns metadata, authorization, retention, and playback shell; plugins may own protocol-native capture when generic browser capture is insufficient.
