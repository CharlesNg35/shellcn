## Goal
Record stream traffic at the core wrapper boundary without making SSH, VNC, or
RDP special cases. The wrapper decides whether recording is active from plugin
capability + connection policy + user permissions.

## Policy semantics (auto = forced)
- `disabled` â€” never record; no tap installed.
- `auto` â€” **forced**: recording starts automatically when the recordable stream
  opens and the user cannot stop it. If a forced recorder cannot start, the stream
  is denied before the upstream session opens.
- `manual` â€” a recording tap is installed but idle; the user starts/stops it
  through control endpoints. Stopping finalizes the recording.
