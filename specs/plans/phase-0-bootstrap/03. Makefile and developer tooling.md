## Checklist

      `test`, `lint`, `gen` (go generate)
- [ ] `golangci-lint` config + `gofumpt`
- [ ] Go live-reload (`air` or `wgo`) for `dev-api`

- [ ] CI workflow stub: build + lint + test

## Definition of done
