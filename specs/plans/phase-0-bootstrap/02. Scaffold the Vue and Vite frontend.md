# Phase 0 · Step 2 — Scaffold the Vue + Vite frontend

**Milestone:** — (bootstrap) · **Status:** ☐ Not started
**Depends on:** 0.1 · **Spec refs:** [v2 §11](../../v2.md), [v2 §14](../../v2.md)

## Goal

Stand up the Vue 3 + Vite + TypeScript app inside `web/` (Vite root = `web/`),
building to `web/dist` so the existing `web/embed.go` embeds it.

## Checklist

- [ ] Scaffold Vite (vue-ts) into `web/` **without** clobbering `web/embed.go`
- [ ] Add deps: `pinia`, `vue-router`, `@vueuse/core`, `primevue` (unstyled/pass-through),
      `tailwindcss`, `xterm`, `@novnc/novnc`, `monaco-editor`
- [ ] Configure Tailwind + PrimeVue pass-through theme tokens
- [ ] `vite.config.ts`: `build.outDir = "dist"`, correct `base` for embedded serving
- [ ] **Code-splitting** set up: heavy panel libs (`monaco-editor`, `@novnc/novnc`, `xterm`, charts) are dynamic imports / separate chunks, **not** in the main bundle (v2 §12.1)
- [ ] Vitest + Playwright wired (config + a smoke test) so testing exists from day one
- [ ] Placeholder `App.vue` renders; TS `strict` enabled
- [ ] Confirm `web/dist`, `web/node_modules`, `web/.vite` are gitignored (already are)

## Definition of done

`npm run build` emits `web/dist` with heavy libs in separate lazy chunks; `npm run dev` serves with HMR; `go build` embeds the built assets.

## Tests (gate)

- [ ] `vitest run` and `playwright test` execute (even if just a smoke test)
- [ ] Build report shows Monaco/noVNC/xterm are **not** in the initial chunk

## Notes

`web/` is simultaneously the Go `web` package (embed.go) and the Vite project root — that's intended. See [`../TESTING.md`](../TESTING.md).
