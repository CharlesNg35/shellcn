## Goal

Stand up the Vue 3 + Vite + TypeScript app inside `web/` (Vite root = `web/`),
building to `web/dist` so the existing `web/embed.go` embeds it.

## Checklist

- [x] Scaffold Vite (vue-ts) into `web/` **without** clobbering `web/embed.go`
- [x] Add deps: `pinia`, `vue-router`, `@vueuse/core`, `primevue` (unstyled/pass-through),
      `tailwindcss` (v4), `@xterm/xterm`, `@novnc/novnc`, `CodeMirror`
- [x] Configure Tailwind + PrimeVue pass-through theme tokens (`tailwindcss-primeui`)
- [x] `vite.config.ts`: `build.outDir = "dist"`, `base = "/"` for embedded serving
- [x] **Code-splitting** set up: heavy panel libs (`CodeMirror`, `@novnc/novnc`, `@xterm/xterm`) are dynamic imports / separate chunks, **not** in the main bundle
- [x] Vitest + Playwright wired (config + a smoke test) so testing exists from day one
- [x] Placeholder `App.vue` renders; TS `strict` enabled (via `@vue/tsconfig`)
- [x] Confirm `web/dist`, `web/node_modules` are gitignored (already are)

## Definition of done

`npm run build` emits `web/dist` with heavy libs in separate lazy chunks; `npm run dev` serves with HMR; `go build` embeds the built assets.
